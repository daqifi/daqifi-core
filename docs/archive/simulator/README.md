# Simulator Planning (Archived)

**Status:** Archived. The planning documents in this directory describe a C# native device simulator that was scoped in 2025 (issues [#61](https://github.com/daqifi/daqifi-core/issues/61)–[#66](https://github.com/daqifi/daqifi-core/issues/66)) but never implemented. The simulator's intended role — hardware-independent integration coverage of real devices — was superseded in 2026 by the hardware-in-the-loop smoke test introduced in [#219](https://github.com/daqifi/daqifi-core/pull/219), so the planning work was closed without code.

They are kept here for reference in case the simulator is ever revived; nothing under `src/` corresponds to them today.
